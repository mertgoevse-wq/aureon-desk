# Vibeforge Simple Home Experience

This document details the design and implementation of the simplified starting screen for Vibeforge.

## Simplicity First
Vibeforge's Home experience prioritizes immediate action and clean visual spacing:

1. **Centered Branding:** The Vibeforge logo and header are centered to draw focus.
2. **One Large Composer:** A single prominent text composer invites the user to describe their idea in plain language.
3. **Curated Examples:** Six distinct example cards represent the most common builder goals:
   - *Build a landing page*
   - *Build a web app*
   - *Build an Android app prototype*
   - *Improve my UI*
   - *Fix a bug*
   - *Explain my project*
4. **Transition to Wizard:** Clicking any example pre-selects the corresponding builder configurations, presents a visual mockup descriptor, and switches to the Step-by-Step Goal Assistant wizard.
5. **No Clutter:** Technical configuration details, advanced options, and debugging logs are hidden behind collapsible containers or secondary tabs.
